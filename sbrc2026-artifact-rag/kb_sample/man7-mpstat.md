# mpstat(1) â€” Linux manual page (excerpt)

`mpstat` reports per-CPU utilization, including `%irq` and `%soft`:

- `%irq`: time servicing hardware interrupts
- `%soft`: time servicing software interrupts (softirqs)
- `%steal`: involuntary wait (VM)

See also: `pidstat(1)`, `sar(1)`, `vmstat(8)`.
