                    Exynos4 boards (``nuri``, ``smdkc210``)
=======================================

These are machines which use the Samsung Exynos4210 SoC, which has Cortex-A9 CPUs.

``nuri`` models the Samsung NURI board.

``smdkc210`` models the Samsung SMDKC210 board.
