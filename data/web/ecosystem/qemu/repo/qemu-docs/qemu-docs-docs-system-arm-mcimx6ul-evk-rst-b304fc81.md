                    NXP MCIMX6UL-EVK (``mcimx6ul-evk``)
===================================

The ``mcimx6ul-evk`` machine models the NXP i.MX6UltraLite Evaluation Kit
MCIMX6UL-EVK development board. It has a single Cortex-A7 CPU.
