                    NXP i.MX25 PDK board (``imx25-pdk``)
====================================

The ``imx25-pdk`` board emulates the NXP i.MX25 Product Development Kit
board, which is based on an i.MX25 SoC which uses an ARM926 CPU.

Emulated devices:

- SD controller
- AVIC
- CCM
- GPT
- EPIT timers
- FEC
- RNGC
- I2C
- GPIO controllers
- Watchdog timer
- USB controllers
